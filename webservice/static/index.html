WebService - Stock Prediction

Available Endpoints:

All endpoints are prefixed with: http://127.0.0.1:5000

Forecast for one day

GET METHOD
  • /prediction/get/api/alpha
    • Ex: /prediction/get/api/alpha?symbol=AAPL
  • /prediction/get/api/iex
    • Ex: /prediction/get/api/iex?symbol=AAPL
POST METHOD
  • /prediction/post
    • Ex: { "symbol": "AAPL", "close":[ "174.8700", "174.3300", "174.2300", "172.9700", "171.0600", "172.0300", "170.9300" ] }

Forecast for multiple days

GET METHOD
  • /prediction/get/api/alpha/days
    • Ex: /prediction/get/api/alpha/days?symbol=AAPL&days=7
  • /prediction/get/api/iex/days
    • Ex: /prediction/get/api/iex/days?symbol=AAPL&days=7
POST METHOD
  • /prediction/post/days
    • Ex: { "symbol": "AAPL", "days": "7", "close":[ "174.8700", "174.3300", "174.2300", "172.9700", "171.0600", "172.0300", "170.9300" ] }
